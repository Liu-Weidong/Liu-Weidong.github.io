Java程序运行机制和JVM

java源文件 –javac编译–> *.class字节码文件（与平台无关，面向JVM） –java解释执行–> 特定平台的机器码（先编译、后解释）
计算机高级语言按程序执行方式分为编译型和解释型两种。

  • 编译型语言指使用专门的编译器，针对特定平台将某种高级语言源代码一次性“翻译”成该平台硬件执行的机器码，并包装成该平台所能识别的可执行性程序的格式，这个转换过程称为编译。（C、C++、Objective-C、Pascal）
  • 解释型语言指使用专门的解释器对源程序逐行解释成特定平台的机器码并立即执行的语言。效率低，跨平台。（Ruby、Python）

JVM
负责解释执行字节码文件的Java虚拟机。JVM就是转换器，分为向上和向下两个部分。
